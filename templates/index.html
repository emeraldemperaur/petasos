Built on a secure PostgreSQL DB

Petasos /peh· tuh· zowz/ is a bespoke Flask-RESTful microservice built to receive destination or place data requests from an authenticated client (mobile, web, hybrid, OS, Embedded) application and return appropriate response results pertinent to the client HTTP request method - DB data privately curated from destination cities & places of interest across the globe.

Petasos API Endpoints

GET, POST, PUT, DELETE HTTP request method(s)

/destination/string:name GET, POST, PUT, DELETE
/place/string:name GET, POST, PUT, DELETE
/destinations GET only
/places GET only
/destinationnames GET only

JSON OBJECT RESPONSES

Respective endpoints return a variety of mutable JSON objects containing all the Petasos destinations data on the Hermes-Venus PostgreSQL database, including associated places information as well as place type, currency and current weather ℃ for data filtering

JWT AUTHENTICATION ENABLED

JSON Web Tokens implemented to facilitate secure API User Authentication and Log In
